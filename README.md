# Java Bank CLI

Sistema simples de banco no terminal feito em Java, focado em POO.

## Funcionalidades
- Criar conta
- Listar contas
- Depositar
- Sacar (com validações)
- Transferir entre contas

## Conceitos usados
POO, ArrayList, encapsulamento, validações, menu no terminal.

## Como rodar
1) Compile:
javac src/*.java

2) Rode:
java -cp src Main
